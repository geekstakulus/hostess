ID: cap1Languages: O1 O2 AORelevance: standard
Type: run [Input]

Tests CAP over the whole ASCII range, that is, for the character codes 0..127. Except for a..z, all characters should be mapped onto themselves.

ID: cap2Languages: O1 O2 AORelevance: standard
Type: run [Input]

Tests CAP over the whole ISO Latin1 range 0..127. Except for characters that have a corresponding upper case letter in ISO Latin1, all characters should be mapped onto themselves.

ID: with1Languages: O1 O2 AORelevance: standard
Type: accept [Input]

Recursive call to a procedure from within a WITH statement. At the place of the call, the WITH is applied to the procedure's argument. Compilers that modify the external signature of the procedure for the duration of a WITH guard reject this test.

ID: procvar1Languages: O1 O2 AORelevance: standard
Type: accept [Input]

If this test passes, then the compiler applies the same relaxed type rules to the comparison of procedure values as to the assignment of procedure values.
